# Daily Brief · 全球政经科技日报

每日自动生成的全球情报简报，涵盖政治、经济、科技三大领域。

## 📰 在线阅读

**今日简报**: [https://wangshuang12138-beep.github.io/daily-brief/](https://wangshuang12138-beep.github.io/daily-brief/)

**历史归档**: [https://wangshuang12138-beep.github.io/daily-brief/archive/](https://wangshuang12138-beep.github.io/daily-brief/archive/)

## 📋 内容板块

- **📰 今日要闻 HEADLINES** - 国际政治、宏观经济、科技前沿各 3 条精选
- **🔍 深度解读 IN-DEPTH** - 选一个重点话题展开分析
- **📖 英语角 ENGLISH CORNER** - 精选英文原文 + 中文词汇注解
- **📊 数据速览 DATA BRIEF** - 股市、汇率、加密货币等关键数据

## ⏰ 更新时间

每天北京时间 **08:00** 自动更新

## 🏗️ 技术架构

- **生成**: Python 脚本 + GitHub Actions
- **部署**: GitHub Pages
- **样式**: 复古报纸风格 CSS
